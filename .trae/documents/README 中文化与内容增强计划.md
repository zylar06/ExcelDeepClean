## 2. 内容大纲

### 标题与简介
- 项目名称：ExcelDeepClean (智能 Excel 表格提取与清洗)
- 简介：将复杂、非结构化的 Excel 报表转化为高可用结构化数据的 AI 驱动工具。

### 核心优势 (Why Use This?)
- **突破限制**：解决合并单元格错位、多表混排、层级标题等传统解析痛点。
- **智能语义**：利用 LLM 自动识别“个案”、“索赔单”等逻辑子表，无需人工配置规则。
- **极低资源**：流式读取技术，支持 10w+ 行大文件，内存占用极低。

### 处理流程 (How It Works?)
#### 阶段一：物理提取与标准化 (Rule-based Extraction)
- **解决问题**：
  - 合并单元格导致的数据丢失（实现 Forward Fill）。
  - 多个表格混在一个 Sheet 中（实现连通域检测）。
  - 空白列占位（实现自动修剪算法）。
- **技术点**：`xml` 流式解析、连通域算法。

#### 阶段二：智能语义清洗 (AI-driven Refinement)
- **解决问题**：
  - 逻辑分表：自动识别“系统信息”、“备注”等分节标题并切分为独立 Sheet。
  - 噪声清洗：剔除“新建”、“更改所有人”等无意义操作行。
  - 冗余表头：智能合并内容重复的列（如“排序”）。
- **技术点**：Data Profiling、Feature Sampling、Prompt Engineering。

### 快速开始 (Quick Start)
- 安装依赖 (`uv sync`)。
- 配置 API Key。
- 运行命令示例。
